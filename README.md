# MtaaniWeb

**Your Journey to Online Success**

MtaaniWeb is a done-for-you web presence service for small businesses 
and hustlers in Kenya who need to get online fast and affordably.

## What This Site Is

This is the official MtaaniWeb website. It is a single-file HTML site 
hosted on GitHub Pages, covering all pages including Home, About, 
Contact, Reviews, Blog, Terms and Conditions, and Privacy Policy.

## Services

**Starter Sites** -- Up to 3 pages, custom .co.ke domain, optional 
business email, mobile-friendly design, hosted and managed by MtaaniWeb. 
Renewed yearly.

**Business Systems** -- Simple digital systems for small businesses moving 
from analogue to digital. Booking management, staff portals, reporting, 
and more. Scoped and priced per project.

## Tech Stack

- Single-file HTML with embedded CSS and JavaScript
- Hosted on GitHub Pages
- Domains registered via HostPinnacle
- Email hosting via Host Africa
- App-level projects use Firebase Firestore and Authentication
